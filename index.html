Dataset

My Dataset was taken from Kaggle (https://www.kaggle.com/datasets/rashikrahmanpritom/disney-movies-19372016-total-gross)

Walt Disney Studios Movie Information: Gross is a significant indicator of movie performance. This data frame consists of 579 Disney movies and their corresponding details from 1937 through 2016. Analyzing the Disney Movies' Total Gross dataset over time will yield insight into the results and visualizations for the following questions:

  1. Top 10 highest grossing Disney movies adjusted for inflation
  2. Top 10 lowest grossing Disney movies adjusted for inflation
  3. Top 10 highest grossing Disney movies in the 21st Century adjusted for inflation
  4. Top 10 highest grossing Disney musicals adjusted for inflation
  5. In which decade did Walt Disney Studios films have the highest average gross adjusted for inflation
  6. Which genre of film has the highest average gross adjusted for inflation
  7. Which MPAA rating type has the highest average gross adjusted for inflation
  8. Is there a correlation between length of movie title and gross adjusted for inflation
  9. How many movies has Disney produced under each genre category

5. Which Decade did Disney Movies Gross the Highest on Average (Inflation Adjusted)

To organize the data into decades, we establish the bounds to determine the number of decades we will look at. Using the Arange method, we generate an array containing the starting year of each decade. This method takes a start index, an end index, and a step value to return an array containing the year at the beginning of each decade.

We use this array to create a tuple where the zero position contains the array's decade start value, and the one position in the tuple gets the end bound for the decade by adding ten to the start.

This tuple allows us to generate a bin with specific ranges of index values using the IntervalIndex method.

We then plug the release_year column values into the pandas cut method to compare to the bin value, and if the value is within a decade's range, the entry gets that particular decade. As a result, we can add a new decade column storing the corresponding decade for each entry.

The bin labels in the form [start, end) constitute the decade column values following this code snippet. However, we replace these values with cleaner labels using a dictionary replace, which checks if DataFrame values match any of its keys and replace them with their respective values if found.

In[19]:
start_year = 1930
end_year = 2017
decades = np.arange(start_year, end_year, 10).tolist()

tuples = [(start, start + 10) for start in decades]
bins = pd.IntervalIndex.from_tuples(tuples, closed = "left")

data["decade"] = pd.cut(data["release_year"],
                        bins = bins,
                        include_lowest = True)

bin_labels = list(bins.astype(str))
decade_labels = ['{}s'.format(str(d)) for d in decades]
dictionary = dict(zip(bin_labels, decade_labels))

data["decade"] = data["decade"].astype(str)
data = data.replace(dictionary)

decade_data = data.reset_index().filter(["inflation_adjusted_gross", "decade"], axis = 1).groupby("decade").mean()

6. In Which Genre of Film does Disney Gross Highest on Average (Inflation Adjusted)

The following code copies the genre and inflation gross columns from the original data set and adds them to another. This new data frame groups the entries into their respective genres and calculates the mean gross of each genre. The average gross of a genre can be calculated by summing the inflation-adjusted gross of every movie in that genre and dividing it by the number of addends. The mean() method simplifies this process. The outputted gross is written in scientific notation.

In[22]:
top_gross_genre = data.dropna(
